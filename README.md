# RMS V2
 Online restaurent reservation System using Node js
## Install
download the repo and import the sql and then run ** npm i **

## Features

- Admin Dashboard
- Restaurant add
- Menu add
- Memeber add
- CRUD on every restaurant and to their menus
- CRUD members
- **Validation**
- ** Password hasing **

## Language used
- ** Express js **
- ** MySql **
